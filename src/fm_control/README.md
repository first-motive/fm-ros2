# fm_control

ros2_control hardware interfaces. C++. Drives hardware from orchestration commands.

## Role

```
fm_orchestration -> fm_control (ros2_control) -> hardware
```

## Run

```bash
ros2 run fm_control control_node
```

`control_node` is a stub — replace with the real `hardware_interface` plugin.

## Build type

`ament_cmake` (C++17).
